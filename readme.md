## The Game

* Click the circles
	* Hit the circles withhin a time frame, get as many as you can in a row!
* Test your aim
	* Adjustable settings for:
	  * Speed
	  * Circle Size

## How To Use

To play the game, compile, and run the [Main.java](Main.java) file.
